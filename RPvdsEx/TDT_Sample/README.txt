stream2.rpx
	ストリーム知覚の刺激呈示プログラム
	高音，低音のストリームの中心周波数を一定周期で変化させる
	全体の音の平均：CenterF(1000Hz)
	ストリームの中心周波数の分布中心のCenterFとの比率，変化の周波数：Ratio(2.0)，Fcenter(0.05Hz)
	ストリーム内の音の変化幅：Variance(0.05)
	とすると，
	高音，低音ストリームの中心周波数：CenterH, CenterL
	実際に発生する音の周波数：f
	は以下の式で算出している．
	CenterH = CenterF･ratioH
	CenterL = CenterF･ratioL
	f = Center･(1+rand(Variance))  (Centerは音によってCenterHまたはCenterL)
	但し，
	ratioH = Fcenter･(range･sin(2π･Fcenter･t) + (1.0+range))
	ratioL = 1/ratioH
	range = 0.5(Ratio-1)
	rand(x)は[-x, x]の擬似白色乱数
	2004/3/8

	ratioHの位相が第4象限から第1象限に入るときに，1000msのトリガ(1)を出力するようにした．
	トリガのあと，ストリームは分離し，融合して周波数Fcenter[Hz]で元に戻る．
	2004/3/15
